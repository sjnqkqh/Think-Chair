# 수업자료 점검 기준

## 독자
- [ ] 대상 독자의 수준(초급/중급/고급)이 명시되었는가
- [ ] 선수 지식이 명시되었는가

## 내용
- [ ] 학습 목표가 문서 상단에 명시되었는가
- [ ] 새 용어 등장 시 반드시 설명이 포함되었는가
- [ ] 개념 → 예제 → 실습 순서를 따르는가
- [ ] 난이도가 대상 독자 수준에 적합한가
- [ ] 요약/복습 섹션이 포함되었는가

## 예제
- [ ] 예제가 개념 이해에 직접 기여하는가
- [ ] 예제가 실행 가능한가
- [ ] 충분한 수의 예제가 있는가 (개념당 최소 1개)

## 형식
- [ ] 제목/소제목이 계층적으로 구성되었는가
- [ ] 코드 블록에 언어 지정이 되었는가
- [ ] 한자가 사용되지 않았는가
- [ ] 불필요한 ASCII 그림이 없는가

## 판정
- ✅ 양호: 미달 0개
- ⚠️ 보완 필요: 미달 1-2개
- ❌ 재작성 필요: 미달 3개 이상
