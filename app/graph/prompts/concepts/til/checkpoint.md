# TIL 점검 기준

## 내용
- [ ] 오늘 배운 내용에 집중하는가 (과거 경험 회고 금지)
- [ ] 핵심 내용이 정확한가
- [ ] 불필요한 설명/부연이 없는가
- [ ] 배운 내용과 이해한 내용이 명확히 구분되는가

## 형식
- [ ] 간결하게 정리되었는가 (딥다이브 스타일 금지)
- [ ] 코드/예제가 있다면 짧고 명확한가
- [ ] 맞춤법과 띄어쓰기가 올바른가
- [ ] 문장 구조가 정리되었는가

## 금지 확인
- [ ] 개인적 감상/회고가 포함되지 않았는가
- [ ] 추가 자료 추천이 포함되지 않았는가
- [ ] 한자가 사용되지 않았는가
- [ ] "오늘은 ~에 대해 배웠다" 같은 군더더기 도입이 없는가

## 판정
- ✅ 양호: 미달 0개
- ⚠️ 보완 필요: 미달 1개
- ❌ 재작성 필요: 미달 2개 이상 (TIL은 간결함이 생명)
